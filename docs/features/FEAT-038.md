# Crew Messaging

**ID:** FEAT-038 | **Module:** MOD-008 | **Status:** draft | **Priority:** high

SMS-based messaging for crew assignments, changes, and rain delays with delivery confirmation.

**Owner:** Product Manager

## PRD Summary

### Problem
Crew members need timely notifications about assignments, schedule changes, and weather delays. SMS ensures messages reach crew even when they're on the course.

### Goals
1. Send assignment notifications
2. Communicate schedule changes
3. Alert rain delays
4. Confirm message delivery
5. Support broadcast and individual messages

### Success Metrics
- **Message delivery rate:** 99%
- **Crew acknowledgement rate:** 90%

### Scope
**In Scope:**
- SMS delivery
- Assignment notifications
- Change alerts
- Rain delay messages
- Delivery tracking
- Broadcast messaging

**Out of Scope:**
- Group chat features
- Media attachments
- Message threading

### Dependencies
- Twilio integration
- Assignment system
- Weather integration
