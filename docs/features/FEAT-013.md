# Crew & Templates

**ID:** FEAT-013 | **Module:** MOD-003 | **Status:** draft | **Priority:** high

Crew definitions, role assignments, and reusable task templates for efficient scheduling.

**Owner:** Product Manager

## PRD Summary

### Problem
Scheduling requires understanding crew composition, roles, and common task patterns. Reusable templates speed up recurring work assignments.

### Goals
1. Define crews with roles and skills
2. Create reusable task templates
3. Support template-based scheduling
4. Track crew capabilities

### Success Metrics
- **Time to create daily schedule:** Reduced by 60%
- **Template reuse rate:** 80%

### Scope
**In Scope:**
- Crew definitions
- Role management
- Task templates
- Skills tracking
- Template library

**Out of Scope:**
- Complex skill matrices
- Automated crew optimization
- Cross-facility crew sharing

### Dependencies
- Staff management system
- Task definition system
