# Minimal Job Board

**ID:** FEAT-016 | **Module:** MOD-003 | **Status:** draft | **Priority:** medium

Simple digital job board with printable wall view for crew visibility.

**Owner:** Product Manager

## PRD Summary

### Problem
Crews need clear visibility into daily assignments. A simple job board with print capability supports both digital and physical workflows.

### Goals
1. Provide clear daily assignment view
2. Support printable wall display
3. Show task details and priorities
4. Enable status updates

### Success Metrics
- **Crew understanding of daily tasks:** 4.5/5.0
- **Print usage rate:** 60%

### Scope
**In Scope:**
- Daily job board view
- Printable format
- Task details
- Status indicators
- Priority marking

**Out of Scope:**
- Multi-day calendar view
- Drag-and-drop scheduling
- Mobile-optimized board

### Dependencies
- Assignment system
- Printing infrastructure
- Task definitions
