# Field Capture

**ID:** FEAT-020 | **Module:** MOD-004 | **Status:** draft | **Priority:** medium

Mobile phone-based area tracing and quick map corrections while walking the course.

**Owner:** Product Manager

## PRD Summary

### Problem
Course features change over time and initial maps may have inaccuracies. Field capture enables superintendents to trace boundaries and make corrections on-site.

### Goals
1. Enable GPS-based area tracing
2. Support quick boundary corrections
3. Capture while walking
4. Sync updates to main map

### Success Metrics
- **Field update adoption rate:** 70%
- **Map accuracy improvement:** 95%

### Scope
**In Scope:**
- GPS tracking for tracing
- Mobile interface
- Boundary capture
- Quick corrections
- Sync to main map

**Out of Scope:**
- Offline map editing
- High-precision surveying
- Automated feature detection

### Dependencies
- Mobile interface
- GPS capabilities
- GIS sync system
- Course mapping
