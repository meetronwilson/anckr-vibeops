# Agronomy KPIs

**ID:** FEAT-043 | **Module:** MOD-009 | **Status:** draft | **Priority:** high

Track applications within optimal windows and re-treatment rates to measure agronomic effectiveness.

**Owner:** Product Manager

## PRD Summary

### Problem
Measuring agronomic success requires tracking application timing accuracy and treatment effectiveness. These KPIs demonstrate value to superintendents.

### Goals
1. Track applications within optimal windows
2. Measure re-treatment rates
3. Monitor trigger response times
4. Analyze treatment effectiveness

### Success Metrics
- **Optimal window application rate:** ≥90%
- **Re-treatment rate:** ≤15%

### Scope
**In Scope:**
- Window compliance tracking
- Re-treatment analysis
- Trigger response metrics
- Treatment outcomes
- Dashboard visualization

**Out of Scope:**
- Predictive disease modeling
- Soil health analytics
- Advanced agronomic modeling

### Dependencies
- Application tracking
- Triggers engine
- Analytics infrastructure
