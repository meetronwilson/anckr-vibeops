# Layer Controls

**ID:** FEAT-021 | **Module:** MOD-004 | **Status:** draft | **Priority:** medium

Toggle and stack different map layers including surfaces, irrigation, and applications.

**Owner:** Product Manager

## PRD Summary

### Problem
Managing complex course information requires viewing different data layers independently and in combination. Layer controls enable focused analysis.

### Goals
1. Toggle individual layers on/off
2. Control layer stacking order
3. Adjust layer opacity
4. Save layer configurations

### Success Metrics
- **User satisfaction with layer management:** 4.5/5.0
- **Time to find information:** Reduced by 50%

### Scope
**In Scope:**
- Layer visibility toggle
- Stacking order control
- Opacity adjustment
- Configuration presets

**Out of Scope:**
- Complex layer filtering
- Custom layer creation by users
- Real-time layer updates

### Dependencies
- Course mapping
- GIS rendering engine
- UI framework
