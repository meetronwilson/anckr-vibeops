# Work Order Matching

**ID:** FEAT-025 | **Module:** MOD-005 | **Status:** draft | **Priority:** medium

Link purchase invoices to application and labor work orders for cost allocation.

**Owner:** Product Manager

## PRD Summary

### Problem
Understanding true costs requires linking purchases to the work orders where products and labor were used. Work order matching enables accurate cost allocation.

### Goals
1. Match invoices to work orders
2. Allocate costs to applications
3. Track labor-related expenses
4. Enable cost analysis by job
