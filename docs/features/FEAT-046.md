# Auth & RBAC

**ID:** FEAT-046 | **Module:** MOD-010 | **Status:** draft | **Priority:** critical

Authentication and role-based access control with Owner/Super/CrewLead/Crew roles and SMS magic codes for crew members.

**Owner:** Product Manager

## PRD Summary

### Problem
Secure access control requires proper authentication and role-based permissions. SMS magic codes enable frictionless crew access without passwords.

### Goals
1. Implement secure authentication
2. Define role hierarchy
3. Enable SMS magic codes
4. Support permission management
5. Maintain security standards

### Success Metrics
- **Authentication security score:** A+ rating
- **Crew login success rate via SMS:** 95%

### Scope
**In Scope:**
- User authentication
- Role definitions
- Permission system
- SMS magic codes
- Session management
- MFA support

**Out of Scope:**
- SSO integration initially
- Biometric auth
- Complex permission inheritance

### Dependencies
- User management
- SMS integration
- Session infrastructure
- MakerKit auth system
