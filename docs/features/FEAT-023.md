# Invoice Inbox

**ID:** FEAT-023 | **Module:** MOD-005 | **Status:** draft | **Priority:** high

Automated parsing of invoices from CSV/XLS/email attachments into draft expense records.

**Owner:** Product Manager

## PRD Summary

### Problem
Manual invoice entry is time-consuming and error-prone. Automated parsing from common formats speeds up expense tracking.

### Goals
1. Parse CSV and Excel invoices
2. Extract email attachments
3. Create draft expense records
4. Match to vendors and products

### Success Metrics
- **Invoice processing time:** Reduced by 80%
- **Data extraction accuracy:** 95%

### Scope
**In Scope:**
- CSV parsing
- Excel parsing
- Email attachment extraction
- Vendor matching
- Product matching
- Draft creation

**Out of Scope:**
- OCR from paper invoices
- Complex multi-vendor consolidation
- Automated approval workflows

### Dependencies
- File upload system
- Email integration
- Vendor management
- Products catalog
