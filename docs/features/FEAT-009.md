# Recipe Cards

**ID:** FEAT-009 | **Module:** MOD-002 | **Status:** draft | **Priority:** high

Auto-generated, printable application recipe cards saved to jobs for crew reference.

**Owner:** Product Manager

## PRD Summary

### Problem
Crew members need clear, printable instructions for chemical and fertilizer applications including products, rates, mixing, and safety information.

### Goals
1. Auto-generate recipe cards from application plans
2. Provide printable format for field use
3. Include all necessary safety and mixing information
4. Save recipes to job records for compliance
