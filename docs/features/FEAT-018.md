# Course Import

**ID:** FEAT-018 | **Module:** MOD-004 | **Status:** draft | **Priority:** high

Import course layouts from KML/GeoJSON/GPX files to define surfaces like greens, tees, fairways, and rough.

**Owner:** Product Manager

## PRD Summary

### Problem
Course mapping requires digitizing physical layouts. Supporting standard GIS formats enables quick import from Google Earth and other tools.

### Goals
1. Import KML, GeoJSON, and GPX files
2. Parse surface types
3. Create area definitions
4. Validate imported data
