# SMS Acknowledgements

**ID:** FEAT-015 | **Module:** MOD-003 | **Status:** draft | **Priority:** high

Simple START/STOP/DONE time and status updates via text message from crew members.

**Owner:** Product Manager

## PRD Summary

### Problem
Crew members need a frictionless way to update task status and time without requiring app login or complex interfaces.

### Goals
1. Enable SMS-based status updates
2. Track START/STOP/DONE times
3. Support simple command syntax
4. Link updates to assignments
