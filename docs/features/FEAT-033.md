# Create Course Wizard

**ID:** FEAT-033 | **Module:** MOD-007 | **Status:** draft | **Priority:** critical

Guided setup wizard for organization creation, staff import, area definition, and product catalog initialization.

**Owner:** Product Manager

## PRD Summary

### Problem
New users need to set up their course quickly without overwhelming complexity. A guided wizard streamlines the essential setup steps.

### Goals
1. Guide through organization setup
2. Import staff from CSV
3. Define course areas
4. Initialize product catalog
5. Complete setup in ≤2 hours

### Success Metrics
- **Setup completion time:** ≤2 hours
- **Setup completion rate:** 95%
- **User satisfaction with setup:** 4.5/5.0

### Scope
**In Scope:**
- Step-by-step wizard
- Org setup
- CSV staff import
- Area definition
- Product selection
- Progress tracking

**Out of Scope:**
- Advanced configuration
- Bulk multi-course setup
- Automated data migration

### Dependencies
- User management
- Area management
- Products catalog
- CSV parser
